# Build Scripts

Scripts auxiliares para build do projeto (Vivado + Petalinux).

## Scripts

### (Futuro) `build_vivado.sh`
Automatiza synthesis + implementation do Vivado.

### (Futuro) `build_petalinux.sh`  
Automatiza build completo do Petalinux.

### (Futuro) `package_sd_card.sh`
Prepara imagem SD Card completa (boot + rootfs).

---

**Nota**: Por enquanto, use comandos manuais descritos em `docs/PETALINUX_GUIDE.md`.
